# RGitHub

RGitHub. A GitHub Client.

一个 GitHub 客户端，加载内容会缓存，应用未关闭情况下，可能获取到的更新不够及时，但考虑到 GitHub 的特点，这样做是可以接受的。更多内容看 [这里](http://renyuzhuo.cn/#/issues/39)。
